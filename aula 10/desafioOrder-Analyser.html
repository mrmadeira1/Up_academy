Desafio 1 - Ordenar números

Criar um programa que, recebendo no min 3 números do utilizador através de um unico input, os ordena com click num butao por ordem decrescente e mostra a ordem ao user através de um novo elemento html. Não se pode usar o método sort ou reverse do javascript ou outro método que ordene sozinho.

Desafio 2 - Number analyser

Criar um programa que, recebendo uma sequência qualquer de números do utilizador (input - ex: 328322832782387) coloca um “ - “ entre cada dois números pares e um “#” entre cada dois ímpares. E apresenta o resultado num novo elemento html.
